# sangou1270의 블로그

이 저장소는 GitHub Pages와 Jekyll을 사용하여 호스팅되는 개인 블로그입니다.

## 특징

- Jekyll을 사용한 정적 웹사이트
- 반응형 디자인
- 마크다운으로 작성된 포스트
- 카테고리 및 태그 시스템
- 댓글 시스템 (Disqus 사용 예정)

## 로컬에서 실행하기

```bash
# 의존성 설치
bundle install

# 로컬 서버 실행
bundle exec jekyll serve
```
